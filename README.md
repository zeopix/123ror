Configuring proxy
------------------------------
You can use the env var (.env avaliable) PROXIES_LIST with comma separated value proxies list.

Run Tests
------------------------------
Jasmine tests using teaspoon (avaliable on browser too)
```bash
  $ rake teaspoon
```

Rspec tests:
```bash
  $ rspec spec/
```
